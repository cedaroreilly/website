Web Design

I designed and coded the website www.thorconway.com using HTML, CSS, and jQuery. This website was created in order to make the history, legends and aboriginal wisdom of northern Ontario and the upper Great Lakes areas of Canada available to everyone.

A jQuery plugin is used in the gallery pages so that the images are displayed in a "lightbox".

An online store was created so that the books featured on this website can be sold to customers worldwide.
